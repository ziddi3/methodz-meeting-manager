# Version 0.6 Focused Tests

Use this short checklist after opening `meeting.html` in a browser.

## Numbering

- [ ] Confirm Meeting Numbering Settings appears.
- [ ] Confirm the preview shows the next number.
- [ ] Set prefix to `MTG`.
- [ ] Enable year numbering.
- [ ] Set next sequence to `12`.
- [ ] Save settings.
- [ ] Confirm the hero label shows the configured number.
- [ ] Save a meeting.
- [ ] Confirm the next sequence advances.

## Organization Presets

- [ ] Apply `CSW + Method HVAC`.
- [ ] Confirm both organization boxes are checked.
- [ ] Apply `Partner / Guest Review`.
- [ ] Confirm partner and guest boxes are checked.
- [ ] Save a custom preset.
- [ ] Apply the custom preset.
- [ ] Export custom presets.
- [ ] Import custom presets.
- [ ] Delete a custom preset.

## Duplicate Review

- [ ] Save one test meeting.
- [ ] Create or import a second record with the same title and date.
- [ ] Confirm the duplicate group is detected.
- [ ] Export the duplicate report.
- [ ] Open a duplicate from the review panel.

## Sync Readiness

- [ ] Save or update a meeting.
- [ ] Confirm queued changes appear.
- [ ] Mark all records queued.
- [ ] Export a sync package.
- [ ] Open the JSON file and confirm it includes:
  - `records`
  - `duplicateReview`
  - `syncQueue`
  - `numberingSettings`
  - `organizationPresets`
  - `attendeeDirectory`
  - `customTemplates`
- [ ] Clear the queue.
- [ ] Confirm saved records remain.

## Regression

- [ ] Save a normal meeting.
- [ ] Open/edit a saved meeting.
- [ ] Export TXT, JSON, HTML, and all-record JSON.
- [ ] Confirm attachment references and signature audit still save.
- [ ] Confirm no buttons appear in print output where they should be hidden.
