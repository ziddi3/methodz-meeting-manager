# Version 0.4 Architecture Notes

Version 0.4 keeps the original static runtime intact and adds two enhancement scripts plus one stylesheet.

## Added Files

- `features-v04-templates.js`: default templates, custom templates, template import/export, and custom agenda items.
- `features-v04-records.js`: import preview, archive filters, open task dashboard, task CSV export, and readable saved-record details.
- `features-v04.css`: visual styles for the v0.4 panels and tables.

## Storage

- Saved records use `methodzMeetingRecords`.
- Drafts use `methodzMeetingDraft`.
- Custom templates use `methodzMeetingTemplates`.

## Deployment

No framework, package manager, build step, server, or external package was added. The app remains deployable as static files.
