# Version 0.4 Test Notes

After opening `meeting.html`, check these areas:

1. Meeting Templates panel appears and default templates can be applied.
2. A current form can be saved as a custom template.
3. Custom templates can be exported and imported as JSON.
4. A custom agenda item can be added, saved, and restored when a saved meeting is opened again.
5. Saved-record JSON import shows a preview before merge.
6. Saved records can be filtered by status and organization / representative.
7. The Open Task Dashboard lists unfinished tasks from saved records.
8. Open tasks can be exported as CSV.
9. The Details button on a saved record opens a readable review panel.
10. The app still loads with no server, package manager, or build command.
