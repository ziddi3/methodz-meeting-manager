# Version 0.4 Notes

Version 0.4 keeps the app static, offline-first, and dependency-free while adding the next workflow layer.

## Added

- Meeting template panel.
- Default templates in `config.js`.
- Browser-local custom templates.
- Template export and import as JSON.
- Custom agenda items.
- Safer saved-record import preview before merge.
- Saved-record filters by status and organization / representative.
- Open Task Dashboard across saved records.
- CSV export for open tasks.
- Readable saved-record details panel.
- Extra styling in `features-v04.css`.

## Files

- `features-v04-templates.js`
- `features-v04-records.js`
- `features-v04.css`

## Architecture

The v0.4 files are layered on top of the stable offline core. They do not add a framework, build step, package manager, server, or external dependency.

## Next Build Direction

Version 0.5 should focus on attachment references, meeting numbering settings, contact presets, organization presets, and a storage adapter boundary for future cloud sync.
